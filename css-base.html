Designer en CSS

  • link : href="./style.css" type="text/css" rel="stylesheet"
  • . pour les classes
  • #pour les id
  • [attribut] = 'value'

Modifier le texte en CSS

  • color : couleur / rgb(000,000,000) / #000000
  • font-size : px
  • font-family : police de son choix, police de secours
  • font-weight : bold / chiffre
  • text-decoration : underline / line-through
  • text-transform : lowercase / uppercase / capitalize ...
  • font-style : italic
  • link : href="lien police" rel="stylesheet" type="texte/css"
  • text-align : center/ justify / left / right
  • line-height : px

Visuels : image, formes, couleurs

  • width : px
  • height : px
  • box shadow : px px (px) (px) color
  • opacity : 0/1
  • position : relative / absolute / fixed
  • float : right / left
  • z-index : numéro
  • background : linear-gradient(90deg, color 1, color 2, ...)
  • background : url(lien)
  • transform : scale(2) / skewX ou Y (deg) / rotate(deg)
  • audio : controls
  • source : src="source" type="type"

Les bords des éléments

  • boder-color : color / rgb(000,000,000) / #000000
  • style : solid / dotted / dashed / double / groove / ridge / inset / outset
  • border-radius : px
  • border-width : px

Margin et Padding

  • margin : haut droit bas gauche / auto
  • padding : haut droit bas gauche

Variable
