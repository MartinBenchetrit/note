Les éléments pour afficher du texte

  • p = paragraphe
  • h1 jusqu'à h6 = titres et sous titres
  • <!-- Insérer commentaire --> pour que le texte n'apparaisse que dans le fichier code

Images

img : src="lien de l'image" alt="description"

Les liens

a : href="chemin"

a href="#nom" emmene vers div : id="nom"

a : href="#" : lien mort

Listes

ul ou ol (puces ou numéro)

li pour chaque élément de la liste

Formulaire

form : action="action une fois le formulaire rempli"

Les différents boutons

  • input : type="text" paceholder="texte deja écrit" (required)
  • button : type="submit"
  • input : type="radio" name="meme nom que les autres"
  • input : type="checkbox" name="meme nom que les autres

Chaque checkbox ou radio doit etre mis dans un label : for="meme que l'id"
